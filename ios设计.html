邀请规则

费用

每创建1个数字分身需支付 ¥1000，费用包含数字分身制作费和第一年使用费。
成功创建即可获得10万积分（价值¥1000，100积分价值¥1），用于使用平台里其他分身，积分只能自己使用，不可转让或提现。

收益

邀请成功创建分身即可获得激励，有效层级为2层：
（1）每邀请成功创建1个分身，即可获得30%创建费的激励。
（2）被邀请人再邀请成功创建1个分身，还可获得12%激励。
示例：
若Alice某月共计邀请10人成功创建分身，Alice即可获得 ¥3000（税前）。若Alice邀请的这10人当月共计成功邀请20人创建分身，Alice可获得 ¥2400（税前）。那么，Alice当月共计获得奖励 ¥6600（税前）。

用户交互流程

核心使用路径

  1. 1. 用户首次打开APP → 查看引导页了解功能 → 创建账户
  2. 2. 进入首页 → 创建第一个数字分身 → 开始训练（上传文档/对话）
  3. 3. 训练完成后 → 与分身对话测试 → 持续优化训练
  4. 4. 创建多个专业分身 → 根据场景使用不同分身
  5. 5. 查看使用统计 → 管理分身 → 个性化设置

交互设计亮点

  • 卡片式设计：信息层级清晰，易于浏览和操作
  • 进度可视化：实时展示分身训练进度，提升用户掌控感
  • iOS原生体验：毛玻璃、圆角、阴影等符合iOS设计语言
  • 极简配色：黑白灰主色调，符合专业、现代的品牌调性
  • 触手可及：主要操作按钮位于拇指热区，单手可操作

  • • Tailwind CSS 样式框架
  • • Unsplash 高质量图片
